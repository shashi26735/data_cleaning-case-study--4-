## Objective
Performed exploratory data analysis (EDA) on an app store dataset to understand app distribution, ratings, reviews, categories, and install patterns using Pandas.

3. Find the shape of the dataset (number of rows & columns)  
4. Get complete dataset information:
   - Total rows
   - Total columns
   - Data types of each column
   - Memory usage  
5. Get overall statistical summary of the dataset  
6. Find total number of app titles containing the word **‚ÄúAstrology‚Äù**  
7. Find the average app rating  
8. Find total number of unique app categories

10. Find total number of apps with **5-star rating**  
11. Find the average number of reviews  
12. Find total number of **Free** and **Paid** apps  
13. Identify the app with the **maximum number of reviews**  
14. Display top 5 apps with the **highest number of reviews**  
15. Find the average rating of **Free vs Paid** apps  
16. Display top 5 apps with the **maximum number of installs**

- Python

## Key Learnings
- Practical use of Pandas functions like `head()`, `tail()`, `shape`, `info()`, `describe()`

- Ranking apps using reviews and install counts
